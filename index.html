Today I studied such console command:

  • cd - moves to another folder
  • dir - show list of files
  • mkdir - make a directory(тека)
  • rmdir - remove a directory(тека)
  • color - change text color
  • cls - clear console

And such git console commands:

  • git add - add file into commit
  • git commit - make commit
  • git push - upload files on GitHub
